INSTRUCTIONS:

Before running the executable of Bilge, first write whatever you want to be encrypted into the file 
"text.txt" in the same directory with the executable.

When you run the executable you will be greeted with a CLI menu.

After choosing the encryption option, the program will ask you for a seed and a key.

While the seed can be both positive and negative integers, the key HAS TO BE a positive integer.

The sizes of the integers aren't a problem AS LONG AS YOU DON'T FORGET what you chose for your seed and key.
(Obviously don't enter a hundred gazillion, as python has its limits.)

After you enter the seed and the key you chose, the program will encrypt your text and write it back in the file "text.txt"

When you want to decrypt your text, simply put it back into the file "text.txt" and before running the executable please don't forget to save the file "text.txt"
!!! (!!! Because the program also encrypts every space and \n in the file "text.txt", you must always use Ctrl+A before you copy/paste !!!) !!!

When decrypting, enter the seed and the key you had chosen for the encryption and the program will decrypt your text and write it back into the file "text.txt"

!!! ALWAYS CHECK THE SEED AND KEY YOU CHOSE BEFORE PROCEEDING, AS ANY TINY MISTAKE WILL BE CATASTROPHIC !!!
!!! AND AGAIN, DON'T FORGET THAT ANY space AND \n YOU HAVE ENTERED IS IMPORTANT !!!

Thank you for your interest!
